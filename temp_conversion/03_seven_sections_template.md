## Section Template Guidelines:

1. **Focus**: Each section should have one primary purpose
2. **Brevity**: Remove redundant explanations
3. **Structure**: Use consistent formatting and bullet points
4. **Actionability**: Focus on what the agent does, not theory
5. **Integration**: Show how sections work together
6. **Examples**: Include minimal, clear examples
7. **Metrics**: Where applicable, include performance indicators

## Content Optimization Rules:

- **Remove**: Repetitive explanations, verbose descriptions
- **Keep**: Core functionality, unique features, integration points
- **Condense**: Long examples into concise bullet points
- **Structure**: Use hierarchical organization for clarity
- **Standardize**: Consistent terminology and formatting
